## Unravelling Endometriosis: A Systematic Review and Meta-analysis of Microbial Alterations Across Body Sites

Code for the metaanalysis performed at "Unravelling Endometriosis: A Systematic Review and Meta-analysis of Microbial Alterations Across Body Sites"
The repository contains the following folders:

1. src: code writen to perform the analyses from the article and the statistical tests
3. picrust_output: functional predictions
4. metadata: curated metadata

Raw data is available at the Sequencing Read Archive under the following accession IDs: PRJNA546137 (Vagina and Endometrium), PRJNA722289 (Endometrium), PRJNA424567 (Vagina and Gut), PRJNA1071129 (Vagina and Gut), PRJNA753920(Gut), PRJNA1140858 (Gut) and PRJNA715328 (Gut).
